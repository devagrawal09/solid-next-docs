# Lazy Signals

Understanding how SolidJS v2 implements lazy evaluation in its reactivity system.

## Overview

Lazy signals in v2 provide:
- On-demand computation
- Automatic cleanup
- Memory optimization

## Key Concepts

- Deferred execution
- Cleanup management
- Resource optimization

## Benefits

- Reduced unnecessary computations
- Better memory usage
- Improved performance
