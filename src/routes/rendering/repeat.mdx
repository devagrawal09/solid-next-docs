# Repeat

The `<Repeat>` component provides optimized rendering for static lists in SolidJS v2.

## Usage

```tsx
import { Repeat } from 'solid-js/v2';

function StaticList() {
  return (
    <Repeat times={5}>
      {(index) => <div>Item {index}</div>}
    </Repeat>
  );
}
```

## Props

- `times` - Number of times to repeat
- `children` - Render prop for each iteration

## Features

- Optimized for static lists
- No key management needed
- Efficient updates
