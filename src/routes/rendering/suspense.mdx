# Suspense

The `<Suspense>` component provides async boundary handling in SolidJS v2.

## Usage

```tsx
import { Suspense } from 'solid-js/v2';

function AsyncContent() {
  return (
    <Suspense fallback={<div>Loading...</div>}>
      <AsyncComponent />
    </Suspense>
  );
}
```

## Props

- `fallback` - Content to show while loading
- `children` - Async content to render

## Features

- Async boundary management
- Loading state handling
- Error boundary integration
