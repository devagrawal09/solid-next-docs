# Show

The `<Show>` component provides conditional rendering in Solid.

## Usage

```tsx
import { Show } from 'solid-js';

function UserProfile({ user }) {
  return (
    <Show
      when={user}
      fallback={<div>Loading...</div>}
    >
      {(user) => <div>Welcome, {user.name}</div>}
    </Show>
  );
}
```
