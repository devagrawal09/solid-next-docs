# createProjection

`createProjection` allows creating derived views of store data with improved performance.

## Usage

```tsx
import { createStore, createProjection } from 'solid-js/v2';

function TodoApp() {
  const [state] = createStore({
    todos: [],
    filter: 'all'
  });
  
  const filteredTodos = createProjection(
    () => state.todos,
    todos => todos.filter(todo => todo.completed)
  );
  
  return (
    <div>
      {filteredTodos().map(todo => (
        <div>{todo.text}</div>
      ))}
    </div>
  );
}
```

## API

```ts
function createProjection<T, U>(
  source: () => T,
  projection: (value: T) => U
): () => U
```

## Features

- Efficient derived state
- Automatic updates
- Type-safe projections
