# Store Utilities

A collection of utilities for working with stores in SolidJS v2.

## Overview

Available utilities:
- `produce` - For immutable updates
- `reconcile` - For deep state updates
- `unwrap` - For accessing nested state

## Usage

```tsx
import { createStore, produce } from 'solid-js/v2';

function TodoApp() {
  const [state, setState] = createStore({
    todos: []
  });
  
  const addTodo = (text) => {
    setState(produce(state => {
      state.todos.push({ text, completed: false });
    }));
  };
}
```

## Features

- Immutable updates
- Deep state management
- Type-safe utilities
